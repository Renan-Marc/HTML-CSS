Tipos de tags

Quais são os tipos de tag?

<h1>: Usado para fazer o titulo que vai aparecer no site.

<p>: Utilizado para escrever um texto normal (como este por exemplo).

<hr>: (horizontal roon) Essa tag não possui fechamento com /, ela faz uma linha horizontal.

<br>: (break roon) Responsavel por quebrar o texto
(quebrei aqui por exemplo) na area que você deseja.

O que significa Lt e gt utilizados para representar <.>?

lt: LESS THAN

gt: GREATER THAN
